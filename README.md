# AI Study Guide Generator

## Overview

This project is a web application that uses AI to help students study more efficiently. Users can input notes and generate summaries, quizzes, and flashcards.

## Features

* Generate summaries from input text
* Create quiz questions
* Generate flashcards for studying

## Tech Stack

* Python
* Flask
* OpenAI API (or similar AI model)
* HTML/CSS

## How to Run

1. Clone the repository
2. Install dependencies:

   ```
   pip install -r requirements.txt
   ```
3. Run the app:

   ```
   python app.py
   ```
4. Open in browser:

   ```
   http://127.0.0.1:5000/
   ```
